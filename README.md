# AI-Powered Course Creation System

## 📌 Overview

This project automates course creation by analyzing video transcripts and assigning content to syllabus topics using **Zero-Shot Classification**.

## 🔧 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/imayushthakur/ai_course_creator.git
   cd ai_course_creator
   pip install -r requirements.txt
   streamlit run web_app/app.py
   ```
